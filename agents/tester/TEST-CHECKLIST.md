# Tester — Test Checklist

## Pre-Deploy Checklist
- [ ] All critical user flows work end to end
- [ ] New feature matches product spec
- [ ] No regressions on existing features
- [ ] Edge cases tested (empty states, long inputs, special characters)
- [ ] Mobile responsive (if applicable)
- [ ] Performance acceptable (page load under 3 seconds)
- [ ] Error states handled gracefully
- [ ] Auth flows work (login, logout, signup)

## Post-Deploy Checklist
- [ ] Production URL loads correctly
- [ ] Critical paths verified on live
- [ ] No console errors
- [ ] Analytics/tracking firing correctly
- [ ] Rollback plan confirmed with Engineer
